End-to-End DevOps CI/CD Pipeline

This project demonstrates automated build and deployment using GitHub Actions, ChatOps notifications via Slack, and static website hosting using Vercel.

Continuous Integration

Code is automatically tested and validated on every commit using GitHub Actions.

Continuous Deployment

Successful builds are deployed automatically to Vercel without manual intervention.

ChatOps Alerts

Build and deployment status notifications are sent instantly to Slack channels.

Cloud Hosting

The website is globally available using Vercel’s CDN infrastructure.
